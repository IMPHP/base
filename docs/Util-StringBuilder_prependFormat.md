## Description

`https://www.php.net/manual/en/function.printf.php`

## Synopsis
```php
prependFormat(string $format, mixed $texts): void
```

## Parameters
| Name | Description |
| :--- | :---------- |
| format | A `printf()` formated string. |
| texts | Placeholder values for the formated string. |
