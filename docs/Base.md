# Base
____

## Description
Base package for IMPHP containing some core classes and packages.

## Packages
| Name | Description |
| :--- | :---------- |
| [Utilities](Util.md) | Small utility package that is bundled with IMPHP Base |
| [I/O](IO.md) | Small IO package that is bundled with IMPHP Base |
