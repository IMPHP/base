# [I/O](io.md) / [Stream](io-Stream.md) :: getMetadata

## Description

`https://www.php.net/manual/en/function.stream-get-meta-data`

## Synopsis
```php
getMetadata(): im\util\ImmutableMappedArray
```

## Return
Returns the PHP `stream_get_meta_data()` array
as a `MapArray`.
