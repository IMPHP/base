## Description
Checks to see if a value exists in this list.

## Synopsis
```php
public contains(mixed $value): bool
```

## Parameters
| Name | Description |
| :--- | :---------- |
| value | A value to look for. |

## Return
Returns `true` if an occurences of the value
was found or `false` otherwise.
